## ✨ 產品特色

### 🚀 階梯式習慣養成
- **7天啟動計畫** — 回答3個簡單問題，AI 生成個人化啟動任務
- **21天習慣挑戰** — 完成7天後解鎖，三循環設計（基礎→深化→內化）
- **逐步解鎖機制** — Day 1/3/7/14 分級解鎖新功能，降低初期負擔

### 🤖 AI 教練 — 懂你、陪你、引導你
- **四大教練風格**：嚴格軍官 🪖 / 溫暖鼓勵 💚 / 理性分析 🔍 / 幽默夥伴 😄
- **情境感知回應**：週一正向激勵、週五輕鬆語氣、連續打卡肯定、懈怠時鼓勵
- **每日 AI Tip**：智慧快取機制，每小時更新一次
- **對話持久化**：最近10條歷史上下文，跨會話記憶

### 👥 AI 夥伴揪團成長
4種角色 AI 夥伴，各有獨特人格，陪你走完21天：

| 角色 | 定位 | 互動風格 |
|------|------|----------|
| 🧑‍💼 **同行者 ×2** | 和你同時起步，一起摸索 | 「我也是耶！」共情+不確定 |
| ⭐ **過來人 ×1** | 已完成相同任務，分享經驗 | 「我當時也卡在這裡...」溫暖像學長姐 |
| 🌱 **新手 ×1** | 被你影響而開始 | 「你怎麼做到的？」好奇+敬佩 |
| 🧘 **教練** (可選) | 適時引導，全局視角 | 里程碑/提問時介入 |

- **揪團動態消息流**：打卡、分享、鼓勵、里程碑、提問、反思
- **夥伴私聊**：點任意夥伴1對1深度對話
- **夥伴掉鏈子系統** ⭐ — AI 夥伴偶爾漏打卡，增添真實感，你還能為他們加油！

## 🚀 開發指南

### 環境需求
- Xcode 15+
- iOS 16.0+
- Swift 5.9+

### App Group 配置（Widget 數據共享）
1. 在 Xcode 中選擇主 App Target → Signing & Capabilities → 點擊 "+ Capability" → 添加 "App Groups"
2. 添加 App Group ID：`group.com.bestyearplanner`
3. 對 Widget Extension Target 重複上述步驟，使用**相同**的 App Group ID
4. 確認兩個 Target 的 App Group ID 完全一致，否則 Widget 無法讀取主 App 數據

### 編譯運行
```bash
# Clone 此倉庫
git clone https://github.com/wilson710808/Best-Year-Planner-ios-app.git

# 打開 Xcode
open Best\ Year\ Planner.xcodeproj

# 在 Xcode 中：Build (⌘B) → Run (⌘R)
```

## 📖 設計理念

基於 Michael Hyatt《規劃最好的一年》（*Your Best Year Ever*）五大步驟法則：

1. **相信可能** — 打破限制性信念
2. **總結過去** — 回顧經驗、汲取教訓
3. **找到為什麼** — 發掘內在動機
4. **設定目標** — SMART → SMARTER 升級
5. **付諸行動** — 制定執行計劃

**創新點**：
- 將五大步驟濃縮為「3問題 → 7天啟動 → 21天挑戰」的階梯式流程
- AI 教練提供個性化指導，情境感知回應，持續維持動力
- **AI 夥伴揪團**：社交化成長，不再一個人堅持
  - 同行者一起摸索、過來人分享經驗、新手帶來成就感
  - 夥伴掉鏈子系統增添真實感，用戶體驗激勵他人的成就感
