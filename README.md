## ✨ 產品特色

### 🚀 階梯式習慣養成

- **7天啟動計畫** — 回答3個簡單問題，AI 生成個人化啟動任務
- **21天習慣挑戰** — 完成7天後解鎖，三循環設計（基礎→深化→內化）
- **逐步解鎖機制** — Day 1/3/7/14 分級解鎖新功能，降低初期負擔
- **挑戰進度環形圖** — 7天/21天一目了然

### 🤖 AI 教練 — 懂你、陪你、引導你

- **四大教練風格**：嚴格軍官 🪖 / 溫暖鼓勵 💚 / 理性分析 🔍 / 幽默夥伴 😄
- **情境感知回應**：週一正向激勵、週五輕鬆語氣、連續打卡肯定、懈怠時鼓勵
- **每日 AI Tip**：智慧快取機制，每小時更新一次
- **對話持久化**：最近10條歷史上下文，跨會話記憶

### 👥 AI 夥伴揪團成長

4種角色 AI 夥伴，各有獨特人格，陪你走完21天：

| 角色 | 定位 | 互動風格 |
|------|------|----------|
| 🧑‍💼 同行者 ×2 | 和你同時起步，一起摸索 | 「我也是耶！」共情+不確定 |
| ⭐ 過來人 ×1 | 已完成相同任務，分享經驗 | 「我當時也卡在這裡...」溫暖像學長姐 |
| 🌱 新手 ×1 | 被你影響而開始 | 「你怎麼做到的？」好奇+敬佩 |
| 🧘 教練 (可選) | 適時引導，全局視角 | 里程碑/提問時介入 |

- **揪團動態消息流**：打卡、分享、鼓勵、里程碑、提問、反思
- **夥伴私聊**：點任意夥伴1對1深度對話
- **夥伴掉鏈子系統** ⭐ — AI 夥伴偶爾漏打卡，增添真實感，你還能為他們加油！

### 📖 五大步驟深度強化

| 步驟 | 功能 | 說明 |
|------|------|------|
| 1️⃣ 相信可能 | 限制性信念清單 + AI賦能回應 | 勾選10條常見信念，AI即時生成轉化視角 |
| 1️⃣ 相信可能 | 信念轉化Prompt系統 | AI教練+夥伴所有對話自動嵌入信念引導 |
| 1️⃣ 相信可能 | 信念追蹤持久化 | BeliefRecord 完整 CRUD + 信念日曆 |
| 2️⃣ 總結過去 | 年度回顧 + AI經驗萃取 | 4步引導：成就→遺憾→教訓→AI洞察 |
| 3️⃣ 找到為什麼 | 3個動機 + AI動機卡片 | 強制填寫3個原因，連續3天未打卡自動顯示動機提醒 |
| 3️⃣ 找到為什麼 | 動機耗盡提醒 | CheckInService 追蹤，激勵用戶堅持 |

| 4️⃣ 設定目標 | 領先/滯後指標 | 區分可控行動vs結果，專注可控指標 |
| 4️⃣ 設定目標 | 目標上限提醒 | 超過5個活躍目標時精美警告，干預決策 |
| 4️⃣ 設定目標 | 待棄清單 | 「更少但更好」：記錄決定不做的事，騰出空間 |
| 4️⃣ 設定目標 | 里程碑牆 | 記錄重要突破時刻 |
| 4️⃣ 設定目標 | 週期校正 | 月度/季度目標調整 |
| 5️⃣ 付諸行動 | 一鍵批量打卡 + 補卡 | Dashboard 快捷入口 + 反思式補卡 |

### 📊 進度視覺化

- **習慣熱力圖** — GitHub 風格貢獻圖，視覺化堅持程度
- **能量曲線** — 動機水平追蹤，識別高低點
- **挑戰進度環形圖** — 7天/21天一目了然
- **三維度追蹤** — 事業 / 人際 / 成長
- **進階數據分析** — 維度趨勢圖 + 目標完成時間線 + 習慣養成曲線

### 📈 AI 洞察報告

- **週洞察**：AI 彙整本週數據，生成優勢/改進/聚焦建議
- **月洞察**：深度分析月度趨勢、成就、挑戰
- **激勵語句**：AI 生成個人化激勵

### 🎯 AI 任務生成

- 從目標內容 AI 生成個性化任務（4-6個）
- 每個任務含優先級、截止日期
- Fallback 到規則模板，確保離線可用

### 🎨 現代化體驗

- **Dark Mode** — 自適應深色模式 + 語義顏色
- **Widget 擴展** — 鎖屏顯示今日任務（Small + Medium）
- **本地通知** — 每日挑戰提醒 + 連續打卡提醒
- **無干擾模式** — 專注打卡，減少視覺噪音
- **完成慶祝動畫** — 7/14/21天不同動畫慶祝
